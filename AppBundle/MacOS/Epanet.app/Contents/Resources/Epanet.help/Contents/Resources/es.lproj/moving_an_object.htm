Mover un Objeto

 Inicio  Anterior  Siguiente

Para mover un nudo o un rótulo del esquema a otra posición:

1.Seleccionar el nudo o el rótulo.
2.Pulsar de nuevo sobre el objeto, y manteniendo el botón izquierdo del ratón pulsado, arrastrarlo a la nueva posición.
3.Liberar el botón izquierdo del ratón.

Como alternativa, se pueden introducir las nuevas coordenadas X e Y del objeto en el Editor de Propiedades.

 

Cuando un nudo se desplaza, todas las líneas conectadas a él se desplazan igualmente.
