Water Quality Source Editor

 Top  Previous  Next

The Water Quality Source Editor is used to describe the quality of source flow entering the network at a specific node. This source might represent the main treatment works, a well-head or satellite treatment facility, or an unwanted contaminant intrusion. The editor is invoked from the Property Editor when editing a node by clicking the ellipsis button (or hitting the Enter key) when the Source Quality property has the focus.

 

The dialog box contains the following fields:

 

Field

Description

Source Quality

Baseline or average concentration (or mass flow rate) of source - leave blank to remove the source

Time Pattern

ID label of time pattern used to make source quality vary with time - leave blank if not applicable

Source Type

Select either:

·Concentration

·Mass Booster

·Set Point Booster

·Flow Paced Booster

See Water Quality Sources for details.
