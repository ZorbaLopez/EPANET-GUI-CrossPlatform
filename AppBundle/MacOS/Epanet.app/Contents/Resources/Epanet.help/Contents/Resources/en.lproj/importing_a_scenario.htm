Importing a Scenario

 Top  Previous  Next

To import a Project Scenario from a file:

1.Select File >> Import >> Scenario.

2.Use the Open File dialog box that appears to select a scenario file to import. The dialog's Contents panel will display the first several lines of files as they are selected, to help locate the desired file.

3.Click the OK button to accept your selection.

The data contained in the scenario file will replace any existing data of the same kind in the current project.
