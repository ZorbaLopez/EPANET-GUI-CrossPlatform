Working with Projects

 Top  Previous  Next

Project files contain all of the information used to model a network. They come in two varieties, both containing the same information:

·a binary formatted file, usually named with a .net extension, that can only be created by using EPANET's File >> Save (or Save As) command

·a readable text file, usually named with a .inp extension, that can be created outside of EPANET with any text editor or by using the File >> Export >> Network command. The format of this file is described in the EPANET Users Manual.
