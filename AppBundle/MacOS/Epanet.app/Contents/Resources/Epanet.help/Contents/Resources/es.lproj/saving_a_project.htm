Guardar un Proyecto

 Inicio  Anterior  Siguiente

Para guardar un proyecto, manteniendo su nombre actual:

Seleccionar Archivo >> Guardar de la Barra de Menús, o bien pulsar el botón de la Barra de Herramientas Estándar.

 

Para guardar un proyecto utilizando un nombre diferente:

1.Seleccionar Archivo >> Guardar como desde la Barra de Menús.
2.Desde el diálogo Guardar el Proyecto como, seleccionar la carpeta y el nombre de fichero con el cual se desea guardar el proyecto.

 

Los proyectos se almacenan siempre como ficheros binarios .NET. Para guardar los datos de un proyecto en formato ASCII legible, utilizar el comando Archivo >> Exportar >> Red del Menú Archivo.
