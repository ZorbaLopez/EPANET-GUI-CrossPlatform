Edición de un Grupo de Objetos

 Inicio  Anterior  Siguiente

El diálogo de Edición de un Grupo de Objetos se utiliza para modificar las propiedades de un grupo seleccionado de objetos (Ver Seleccionar un Grupo de Objetos).

Para utilizar este diálogo:

1.Seleccionar una categoría de objetos a editar (Nudos de Caudal o Tuberías).
2.Validar la casilla "con" si se desea añadir un filtro para limitar los objetos seleccionados para su edición. Seleccionar en tal caso una propiedad, una relación y un valor para configurar el filtro. Un ejemplo podría ser "con Diámetro igual a 300".
3.Seleccionar el tipo de cambio a realizar - Reemplazar, Multiplicar, o Incrementar.
4.Seleccionar la propiedad a cambiar.
5.Introducir el valor que debe reemplazar, multiplicar o añadirse al valor actual.
6.Pulsar Aceptar para llevar a cabo la edición.
