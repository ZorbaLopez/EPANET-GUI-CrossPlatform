Negative Pressures

 Top  Previous  Next

EPANET will issue a warning message when it encounters negative pressures at junctions that have positive demands. This usually indicates that there is some problem with the way the network has been designed or operated. Negative pressures can occur when portions of the network can only receive water through links that have been closed off. In such cases an additional warning message about the network being disconnected is also issued.

 

Solutions that supply positive demand at negative pressures can be eliminated by utilizing EPANET's pressure driven analysis (PDA) option (see Hydraulic Analysis Options - Demand Model). It requires that values be supplied for a pressure below which no demand is delivered and a pressure above which full demand is delivered. These two pressures can be the same value (such as 0) to find the smallest amount of demand reductions needed to insure that no node delivers positive demand at a negative pressure.
