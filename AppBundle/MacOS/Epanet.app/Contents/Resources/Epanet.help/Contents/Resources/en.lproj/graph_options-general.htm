Graph Options - General

The following options can be set on the General page of the Graph Options dialog form:

Option

Description

Panel Color

Color of the panel that contains the graph

Background Color

Color of graph's plotting area

View in 3D

Check if graph should be drawn in 3D

3D Effect Percent

Degree to which 3D effect is drawn

Main Title

Text of graph's main title

Font

Changes the font used for the main title
