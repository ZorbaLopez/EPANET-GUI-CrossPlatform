Diálogo de Opciones de la Gráfica

 Inicio  Anterior  Siguiente

El diálogo de Opciones de la Gráfica se aplica para personalizar el aspecto de los gráficos X-Y. Para utilizar este diálogo:

1.Seleccionar una de las cinco pestañas del diálogo, las cuales agrupan las siguientes categorías de opciones:

·Generales

·Eje Horizontal

·Eje Vertical

·Leyenda

·Series de Datos

2.Validar la casilla Por Defecto si se desea utilizar las opciones actuales como opciones por defecto para futuros gráficos.
3.Pulsar el botón Aceptar para validar las opciones elegidas.
