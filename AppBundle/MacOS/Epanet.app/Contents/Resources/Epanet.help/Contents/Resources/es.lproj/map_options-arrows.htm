Opciones de Visualización - Flechas

 Inicio  Anterior  Siguiente

La página de Flechas del diálogo de Opciones del Esquema controla cómo se van a visualizar sobre el esquema las flechas indicativas de la dirección del flujo.

 

Estilo Flecha

Selecciona el estilo (forma) de las flechas a visualizar (seleccionar Ninguno para ocultarlas).

 

Tamaño Flecha

Fija el tamaño de las flechas.

 

Escala Mínima

Fija la escala mínima (en %) a partir de la cual deben visualizarse las flechas; a escalas inferiores todas las flechas se ocultarán.

 

Nota:Las flechas de dirección del flujo sólo podrán mostrarse después de haber realizado una simulación con éxito (ver Ejecutar una Simulación).
