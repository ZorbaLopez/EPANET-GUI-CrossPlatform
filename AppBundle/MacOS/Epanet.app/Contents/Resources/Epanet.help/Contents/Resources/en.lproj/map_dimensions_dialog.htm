Map Dimensions Dialog

 Top  Previous  Next

The Map Dimensions dialog form is used to change the dimensions of the Network Map.

 

Item

Description

Lower Left Coordinates

The X,Y coordinates of the lower left point on the map

Upper Right Coordinates

The X,Y coordinates of the upper right point on the map

Map Units

Units used to measure distances on the map. The choices are:

·Feet

·Meters

·Degrees

·None        

Auto-Size Button

Click to have dimensions be computed based on the coordinates

of objects currently included in the map
