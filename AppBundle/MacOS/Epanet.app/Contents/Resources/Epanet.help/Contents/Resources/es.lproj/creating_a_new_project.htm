Crear un Proyecto Nuevo

 Inicio  Anterior  Siguiente

Para crear un nuevo proyecto:

1.Seleccionar Archivo >> Nuevo en la Barra de Menús, o bien pulsar el botón de la Barra de Herramientas Estándar.
2.Antes de crear un proyecto nuevo se preguntará si se desea guardar el proyecto actual (en caso de existir y haber realizado cambios en él).
3.Seguidamente se creará un proyecto nuevo sin nombre, con todas las opciones fijadas en sus valores por defecto.

 

Cuando se arranca EPANET por primera vez se crea automáticamente un proyecto nuevo.

 

Nota: Las unidades por defecto corresponden al Sistema Métrico Internacional (SI) y la fórmula de pérdidas de carga for defecto es la de Darcy-Weisbach. Si desea usar el Sistema Convencional US o una fórmula de pérdidas de carga diferente debería cambiar esos parámetros inmediatamente después de crear un proyecto nuevo. Ambas se pueden cambiar ajustando las Opciones Hidráulicas por Defecto.

 

Nota: Si se va a utilizar un Mapa de Fondo juntamente con la opción de calcular automáticamente la longitud de las tuberías, se recomienda establecer las dimensiones del área de dibujo inmediatamente después de crear el nuevo proyecto (ver Establecer las Dimensiones del Área de Dibujo).
