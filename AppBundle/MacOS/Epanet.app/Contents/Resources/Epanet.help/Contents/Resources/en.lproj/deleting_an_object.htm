Deleting an Object

 Top  Previous  Next

To delete an object:

1.Select the object on the map or from the Data Browser.
2.Either:
A.click the button on the Standard Toolbar,
B.click the same button on the Data Browser,
C.press the Delete key on the keyboard.

 

 

Note:You can require that all deletions be confirmed before they take effect. See the General Preferences page of the Program Preferences dialog box.
