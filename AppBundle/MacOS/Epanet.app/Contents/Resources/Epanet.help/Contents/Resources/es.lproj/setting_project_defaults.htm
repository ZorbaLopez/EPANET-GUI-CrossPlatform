Establecer los Valores por Defecto de un Proyecto

 Inicio  Anterior  Siguiente

Cada proyecto tiene un conjunto de valores por defecto, que serán adoptados a menos que el usuario los modifique. Para fijar los valores por defecto de un proyecto:

1.Seleccionar Proyecto >> Valores por Defecto.
2.Se mostrará seguidamente un diálogo de Valores por Defecto con tres pestañas, una por cada una de las categorías siguientes:

·Identificativos ID por Defecto

·Propiedades por Defecto de Nudos y Líneas

·Opciones Hidráulicas por Defecto

3.Validar la casilla situada en la parte inferior izquierda del diálogo, si se desea mantener las opciones elegidas para futuros proyectos.
4.Pulsar Aceptar para validar los cambios realizados.
