Exportación de un Escenario

 Inicio  Anterior  Siguiente

Para exportar un Escenario de un proyecto a un fichero de texto:

1.Seleccionar Archivo >> Exportar >> Escenario desde la Barra de Menús.
2.En el diálogo Exportar Escenario mostrado, seleccionar las categorías de datos a exportar.
3.Escribir opcionalmente, en el campo reservado para Notas, una descripción del escenario a guardar.
4.Pulsar el botón Aceptar para confirmar todas las opciones.
5.En el diálogo Guardar Escenario Como, seleccionar una carpeta y el nombre del fichero de escenario. Los ficheros de escenario toman como extensión por defecto .SCN.
6.Pulsar Aceptar para completar la exportación.

El escenario exportado puede ser importado de nuevo al proyecto más tarde, como se describe en la página Importación de un Escenario.
