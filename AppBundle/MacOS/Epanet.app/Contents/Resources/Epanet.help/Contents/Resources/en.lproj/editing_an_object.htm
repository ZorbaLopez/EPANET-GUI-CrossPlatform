Editing an Object

 Top  Previous  Next

To edit an object appearing on the map (a node, link or label):

}If the Property Editor is already visible then simply select the object to be edited

}if the Property Editor is not currently visible then either

·double-click the object on the map

or

·right-click the object on the map and select Properties from the pop-up menu

or

·click on the object and then click the Edit button on the Data Browser

„use the Property Editor to edit the object's properties.

 

To edit an object appearing in the Browser:

}select the object from the Data Browser and then either

·click the Browser's Edit button

or

·double-click the item in the Browser

or

·press the Enter key.

„ for nodes, links and labels the Property Editor will appear to edit the object's properties

„ for other objects one of the following special editor forms will appear to edit the object's properties

·Curve Editor

·Time Pattern Editor

·Controls Editor

 

In addition, the following special editor forms are launched from the Property Editor when editing nodes:

·Demand Editor (Junctions only)

·Source Quality Editor
