Running Pressure Driven Analysis

We are now ready to run the pressure driven hydraulic analysis. Change the base demand at node 5 from 200 GPM to 1000 GPM. Once again select Project | Run Analysis (or click the button). You will receive negative pressure warnings after the simulation. It is time to run pressure driven analysis because the specified demand can not be satisfied.

1.Select Options | Hydraulics from the Data Browser.

2.Use the scroll bar to find Demand Model property and switch from demand driven analysis (DDA) to pressure driven analysis (PDA).

3.Set Minimum Pressure property to be 0 (psi) and Required Pressure to be 20 (psi) and leave Pressure Exponent property as 0.5.

4.Make sure the Status Report property is set to be Yes.

5.Select Project | Run Analysis (or click the button) and you will not receive negative pressure warnings anymore.

6.Open the status report by selecting Report | Status and you will see demand reduction reported.

7.Click node 5 and then select Report | Graph (or click the button) to open Graph Selection dialog box. Select Time Series as the graph type and Pressure as the parameter to plot. In the pressure time series plot, you will see pressure is always larger than 0 psi, but can be lower than 20 psi.
