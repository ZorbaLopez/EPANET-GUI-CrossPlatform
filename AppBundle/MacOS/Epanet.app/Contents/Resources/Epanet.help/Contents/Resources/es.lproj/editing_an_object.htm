Editar un Objeto

 Inicio  Anterior  Siguiente

Para editar un objeto desde el Esquema de la Red:

}Si el Editor de Propiedades se encuentra visible, simplemente pulsar con el ratón sobre el objeto deseado

}Si el Editor de Propiedades no se encuentra visible:

·hacer una doble pulsación sobre el objeto

o

·pulsar sobre el objeto con el botón derecho del ratón y elegir Propiedades del menú emergente

o

·pulsar sobre el objeto y a continuación en el botón Editar del Visor de Datos

„usar el Editor de Propiedades para editar las propiedades del objeto seleccionado.

 

Para editar un objeto desde el Visor de Datos:

}seleccionar el objeto en el Visor de Datos y:

·pulsar el botón Editar

o

·hacer doble clic sobre el elemento en la ventana del Visor

o

·pulsar la tecla Intro.

„ para nudos, líneas y rótulos el Editor de Propiedades aparecerá para editar las propiedades del objeto seleccionado.

„ para otros objetos, uno de los siguientes editores especializados aparecerá para editar las propiedades del objeto seleccionado.

·Editor de Curvas

·Editor de Modulaciones

·Editor de Leyes de Control

 

Adicionalmente, los siguientes editores especializados están disponibles en el Editor de Propiedades para nudos:

·Editor de Demandas (Sólo para Nudos de Caudal)

·Editor de Fuentes de Sustancias Químicas
