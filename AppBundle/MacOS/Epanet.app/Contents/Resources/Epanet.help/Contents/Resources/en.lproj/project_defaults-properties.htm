Project Defaults - Properties

 Top  Previous  Next

The Properties page of the Project Defaults dialog form sets default property values for newly created nodes and links. These properties include:

· Elevation for nodes

·Diameter for tanks

·Maximum water depth for tanks

·Length for pipes

·Auto-Length (automatic calculation of length) for pipes

·Diameter for pipes

·Roughness for pipes

When the Auto-Length property is turned on, pipe lengths will automatically be computed as pipes are added or repositioned on the network map. A node or link created with these default properties can always be modified later on using the Property Editor.
