Los Archivos de Proyecto contienen toda la información requerida para definir el modelo una red. Existen dos tipos de archivos de proyecto, teniendo ambos los mismos datos:

·un archivo binario, con extensión .net, que sólo se puede crear usando el comando Archivo >> Guardar (o Cuardar Como).

· un archivo de texto, con extensión .inp, que se puede crear con cualquier editor de texto o usando el comando Archivo >> Exportar >> Red . El formato de este archivo se detalla en el EPANET Users Manual. Tambien se puede consultar la versión 2.00.12 del manual en español en el Manual del Usuario de EPANET.
