Selecting a Printer

 Top  Previous  Next

To select a printer from among your installed Windows printers and set its properties:

1.Select File >> Page Setup.
2.Click the Printer button on the Page Setup dialog form that appears
3.Select a printer from the choices available in the combo box in the next dialog form that appears.
4.Click the Properties button to select the printer's properties (which vary with choice of printer).
5.Click OK on each dialog form to accept your selections.
