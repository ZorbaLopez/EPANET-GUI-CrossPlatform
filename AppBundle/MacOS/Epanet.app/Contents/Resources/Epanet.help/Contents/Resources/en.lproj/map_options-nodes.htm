Map Options - Nodes

 Top  Previous  Next

The Nodes page of the Map Options dialog form controls how nodes are displayed on the Network map.

 

Node Size

Selects node diameter.

 

Proportional to Value

Select if node size should increase as the viewed parameter increases in value.

 

Display Border

Select if a border should be drawn around each node (recommended for light-colored backgrounds).

 

Display Junctions

Displays junction nodes (all junctions will be hidden unless this option is checked).
