El Editor de Demandas

 Inicio  Anterior  Siguiente

El Editor de Demandas se utiliza para asignar las demandas base y sus modulaciones, cuando existe más de un tipo de demanda en un nudo. Este editor es llamado desde el Editor de Propiedades, en Nudos de Caudal, pulsando sobre el botón con puntos suspensivos (o pulsando Intro) que aparece cuando el foco se sitúa en el campo correspondiente a Tipos de Demanda.

 

El editor es una tabla que contiene tres columnas. Cada tipo de demanda se introduce en una nueva fila de la tabla. Las columnas contienen la siguiente información:

Demanda Basedemanda base o demanda media para cada tipo de demanda (dato requerido)
ModulaciónIdentificativo ID de la modulación aplicable para caracterizar la variación de la demanda en el tiempo (dato opcional)
Tipo DemandaTexto utilizado para identificar el tipo de demanda (dato opcional)

La tabla se dimensiona inicialmente a 10 filas. Si se requieren filas adicionales, basta seleccionar cualquier celda de la última fila y pulsar Intro.

 

Nota:Por convención, la demanda situada en la primera fila del editor será considerada la demanda principal del nudo, y aparecerá en el campo Demanda Base del Editor de Propiedades.
