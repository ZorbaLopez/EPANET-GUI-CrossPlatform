Ver el Resumen del Proyecto

 Inicio  Anterior  Siguiente

Para ver un resumen del contenido del proyecto actual:

1.Seleccionar Proyecto >> Resumen.
2.Aparecerá el diálogo Resumen del Proyecto, en el cual se puede introducir o editar el título del proyecto, y añadir también un texto adicional describiendo los aspectos más relevantes del mismo.
3.En el diálogo se muestran también determinados datos globales del mismo, tales como el número de nudos, tuberías, bombas, etc.
