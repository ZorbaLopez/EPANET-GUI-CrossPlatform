El Editor de Leyes de Control

 Inicio  Anterior  Siguiente

El Editor de Leyes de Control es una ventana con un editor de texto, utilizado para editar tanto las Leyes de Control Simples como las Leyes basadas en Reglas. Incorpora un menú contextual estándar para la edición del texto, que se activa pulsando con el botón derecho del ratón en cualquier punto de la ventana. El menú contiene comandos para Deshacer, Cortar, Copiar, Pegar, Eliminar y Seleccionar Todo.
