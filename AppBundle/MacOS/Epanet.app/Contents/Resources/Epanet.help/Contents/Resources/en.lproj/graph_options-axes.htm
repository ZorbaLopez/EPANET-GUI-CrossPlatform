Graph Options - Axes

 Top  Previous  Next

The Horizontal Axis and Vertical Axis pages of the Graph Options dialog form adjust the way that the axes are drawn on a graph.

Option

Description

Minimum

Sets minimum axis value (minimum data value is shown in parentheses). Can be left blank.

Maximum

Sets maximum axis value (maximum data value is shown in parentheses). Can be left blank.

Increment

Sets increment between axis labels. Can be left blank

Gridlines

Selects type of grid line to draw.

Axis Title

Text of axis title

Font

Click to select a font for the axis title.
