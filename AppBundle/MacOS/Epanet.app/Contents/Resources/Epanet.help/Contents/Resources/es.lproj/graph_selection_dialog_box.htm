Diálogo de Selección de la Gráfica

 Inicio  Anterior  Siguiente

El Diálogo de Selección de la Gráfica, se utiliza para seleccionar el tipo de gráfica y su contenido. Las opciones que ofrece dicho diálogo son las siguientes:

Campo

Descripción

Tipo de Gráfica

Selecciona el tipo de gráfica

Magnitud

Selecciona la magnitud a representar

Instante

Selecciona el instante de tiempo (no se aplica para Curvas de Evolución y el Balance de Caudales)        

Tipo de Objeto

Selecciona entre Nudos o Líneas (en los Perfiles Longitudinales y Mapas de Isolíneas solo pueden representarse la magnitudes asociadas a Nudos)

Elementos a Representar

Selecciona los elementos a representar (aplicable solo para Curvas de Evolución y Perfiles Longitudinales) - ver Seleccionar los Elementos a Representar en la Gráfica
