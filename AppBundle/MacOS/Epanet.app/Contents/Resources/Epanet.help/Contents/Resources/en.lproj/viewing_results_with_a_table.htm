Viewing Results with a Table

 Top  Previous  Next

EPANET allows you to view selected database and analysis results in a tabular format:

·A Network Table lists properties and results for all nodes or links at a specific period of time.

·A Time Series Table lists properties and results for a specific node or link in all time periods.

 

Tables can be printed, copied to the Windows clipboard, or saved to file.
