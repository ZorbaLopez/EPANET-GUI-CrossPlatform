Standard Toolbar

 Top  Previous  Next

The Standard Toolbar contains speed buttons for commonly used commands as follows:

 

       open a new project        

       open an existing project        

       save the current project        

       print the currently active window        

       copy selection to clipboard or to file        

       delete the currently selected item        

       find a specific item on the map        

       run a simulation        

       perform a visual query on the map        

       create a new graph view of results        

       create a new table view of results        

       edit options for the currently active view
