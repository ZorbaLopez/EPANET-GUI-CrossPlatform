Previsualizar la Página

 Inicio  Anterior  Siguiente

Para obtener una Vista Previa de la página impresa, correspondiente a la ventana activa actual:

       Seleccionar Archivo >> Vista Previa desde la Barra de Menús.

Se mostrará un formulario de Previsualización, donde puede observarse el aspecto que presentará cada una de las páginas impresas del objeto elegido.
