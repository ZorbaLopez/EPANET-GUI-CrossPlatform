Graph Selection Dialog

 Top  Previous  Next

The Graph Selection dialog form is used to select a type of graph and its contents to display. The choices available in the dialog consist of the following:

Choice

Description

Graph Type

Select a graph type

Parameter

Select a parameter to graph

Time Period

Select a time period to graph (does not apply to Time Series or System Flow plots)        

Object Type

Select either Nodes or Links (only Nodes can be graphed on Profile and Contour plots)

Items to Graph

Select items to graph (applies only to Time Series and Profile plots) -- see Selecting Items to Graph
