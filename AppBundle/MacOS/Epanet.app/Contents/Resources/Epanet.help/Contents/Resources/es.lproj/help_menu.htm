 Inicio  Anterior  Siguiente

El Menú de Ayuda contiene los comandos dirigidos a obtener la ayuda requerida durante el uso de EPANET. Estos son:

Comando

Descripción

Temas de Ayuda

Muestra una ventana con los temas de ayuda de la aplicación

Unidades

Lista las unidades de medida para todas las magnitudes utilizadas en EPANET

Tutorial

Ofrece un breve tutorial sobre el uso de EPANET

Acerca de

Proporciona información sobre la versión de EPANET en uso

 

Se puede también acceder a la ayuda correspondiente en diferentes contextos, presionando la tecla F1.
