El Editor de Curvas

 Inicio  Anterior  Siguiente

El Editor de Curvas permite introducir las propiedades de una curva de comportamiento. Para utilizar este editor:

1.Introducir los valores de los siguientes campos:

Campo

Descripción

ID Curva

Identificativo ID de la curva (un máximo de 31 caracteres numéricos o alfanuméricos)

Descripción

Texto opcional indicativo de lo que representa la curva

Tipo de Curva

Tipo de curva

Datos X-Y

Datos X-Y de los diferentes puntos de la curva
2.Conforme se van introduciendo datos X-Y en las celdas de la tabla (o pulsando la tecla Intro) es posible ver una representación gráfica de los puntos introducidos en la ventana del editor.
3.Pulsar el botón Aceptar para consolidar la curva.

 

También se puede cargar una curva de comportamiento previamente almacenada en un fichero con el botón Cargar, o almacenar la curva actual en un fichero con el botón Guardar.
