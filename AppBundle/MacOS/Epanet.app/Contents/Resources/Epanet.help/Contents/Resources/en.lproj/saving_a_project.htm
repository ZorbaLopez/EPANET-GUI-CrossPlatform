Saving a Project

 Top  Previous  Next

To save a project under its current name:

Either select File >> Save or click on the Standard toolbar.

 

To save a project using a different name:

1.Select File >> Save As.
2.A standard Save File dialog box will appear from which you can select the folder and name that the project should be saved under.

 

Projects are always saved as binary .NET files. To save a project's data as readable ASCII text, use the File >> Export >> Network command.
