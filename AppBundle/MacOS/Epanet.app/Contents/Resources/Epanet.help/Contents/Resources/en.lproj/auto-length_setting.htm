Auto-Length Setting

 Top  Previous  Next

The Auto-Length setting determines whether pipe lengths are computed automatically whenever a new pipe is created or an existing pipe is repositioned or reshaped on the network map. The setting can be toggled On or Off by either:

·Selecting Project >> Defaults and editing the Auto-Length field on the Properties page of the Defaults dialog form.

·Right-clicking over the Auto-Length section of the Status Bar and then clicking on the popup menu item that appears.

 

Note:To obtain meaningful lengths, the dimensions supplied for the map area must reflect real-world distances. The map length units can be different from the pipe length units -- EPANET handles the conversion internally. See Setting the Map's Dimensions.
