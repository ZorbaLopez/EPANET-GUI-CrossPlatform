Sistema No Resoluble

 Inicio  Anterior  Siguiente

Si en algún momento de la simulación, el sistema de ecuaciones que equilibra caudales y energías para toda la red no puede resolverse, se emitirá el Error 110. Esto ocurre cuando algún sector de la red presenta una demanda, y no existen líneas físicas que lo conecten con ninguna fuente de suministro. En tal caso, EPANET emitirá además mensajes de advertencia para los nudos que queden desconectados. El sistema de ecuaciones puede resultar también irresoluble si se introducen como propiedades de algunos elementos valores poco realistas.
