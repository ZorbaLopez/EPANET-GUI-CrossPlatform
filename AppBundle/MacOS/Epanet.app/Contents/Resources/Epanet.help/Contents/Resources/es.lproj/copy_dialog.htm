Diálogo Copiar

Para elegir cómo y dónde copiar los datos, utilizar el diálogo Copiar, accesible desde la opción de menú Edición >> Copiar a, del siguiente modo:

1.eleccionar un destino para transferir el contenido de la ventana (Portapapeles o Fichero).
2.Seleccionar el formato bajo el cual se desea copiar dicho contenido:

·Mapa de bits (sólo gráficos)

·Metafichero (sólo gráficos)

·Datos (textos, celdas seleccionadas de una tabla, o datos utilizados para construir una gráfica)

3.Pulsar Aceptar para validar todas las opciones elegidas, o Cancelar para abandonar la acción de copiar.

 

El formato Mapa de bits, copia cada uno de los píxeles de una gráfica. El formato Metafichero copia las instrucciones utilizadas para crear la gráfica en formato vectorial, y es más apropiado para pegar la gráfica en documentos como un procesador de textos, donde la gráfica puede reescalarse sin pérdida de resolución. Cuando se utiliza la opción Datos, éstos puede pegarse directamente en una hoja de cálculo para crear tablas o diagramas personalizados.
