Status Report

 Top  Previous  Next

EPANET writes all error and warning messages generated during an analysis to a Status Report. Additional information on when network objects change status and a final mass balance accounting for water quality analyses are also written to this report if the Status Report option in the project's Hydraulics Options was set to Yes or Full.

 

To view a status report on the most recently completed analysis select Report >> Status.

 

The contents of the Status Report can be printed or copied to a file or to the Clipboard (see Printing the Current View and Copying a View).
