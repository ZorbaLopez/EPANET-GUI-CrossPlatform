Graph Options - Legend

 Top  Previous  Next

The Legend page of the Graph Options dialog form controls how the legend is displayed on the graph.

Option

Description

Position

Selects where to place the legend.

Color

Selects color to use for legend background.

Symbol Width

Selects the percent of the label width to use for the symbol width in each legend entry.

Framed

Places a frame around the legend.

Shadowed

Places a drop shadow around the legend.

Visible

Makes the legend visible.

Transparent

Makes the legend background transparent.

Check Boxes

Adds check boxes to each legend entry that allow data series to be included or excluded from the graph.
