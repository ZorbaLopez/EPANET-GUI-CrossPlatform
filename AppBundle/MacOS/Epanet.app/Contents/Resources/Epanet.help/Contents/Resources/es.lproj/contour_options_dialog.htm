Diálogo de Opciones del Mapa de Isolíneas

 Inicio  Anterior  Siguiente

El diálogo de Opciones del Mapa de Isolíneas se utiliza para personalizar el aspecto de los mapas de isolíneas. Las opciones del mismo son:

 

Opción

Descripción

Leyenda

  Mostrar la Leyenda

  Activa o desactiva la visualización de la leyenda

  Modificar la Leyenda

  Fija los colores e intervalos para las isolíneas

Esquema Base

  Líneas

  Color de las líneas que representan el esquema de la red

  Fondo

  Color de fondo para todo el mapa de isolíneas

  Grosor Líneas

  Grosor de las líneas que representan el esquema de la red

Estilo

  Curvas Sombreadas

  Las franjas entre isolíneas se rellenan con un color opaco

  Curvas Simples

  Se dibujan solo las curvas de las isolíneas

Isolíneas

  Grosor

  Fija el grosor de las isolíneas principales

  Líneas por Nivel

  Número de isolíneas a interpolar entre dos isolíneas principales

Casilla por Defecto

Guarda las opciones actuales para futuros mapas de isolíneas
