Formato de las Cláusulas de Acción

 Inicio  Anterior  Siguiente

Una Cláusula de Acción en una Ley de Control basada en Reglas presenta la siguiente sintaxis:

objeto ID STATUS/SETTING IS valor

donde:

objeto

=

la palabra clave LINK, PIPE, PUMP o VALVE

ID

=

el identificativo ID del objeto

valor

=

una condición de estado (OPEN o CLOSED), la velocidad relativa de una bomba, o la consigna de una válvula

 

A continuación se muestras algunos ejemplos de cláusulas de acción:

LINK 23 STATUS IS CLOSED 

PUMP P100 SETTING IS 1.5 

VALVE 123 SETTING IS 90
