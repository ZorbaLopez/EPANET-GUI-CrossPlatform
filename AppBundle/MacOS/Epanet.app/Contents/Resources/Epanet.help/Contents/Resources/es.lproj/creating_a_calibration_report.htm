Crear un Informe de Calibración

 Inicio  Anterior  Siguiente

Para crear un Informe de Calibración:

1.Primero asegurarse de que los Datos de Calibración (o medidas de campo) para la magnitud que se desea calibrar hayan sido registrados en el proyecto (ver Registrar los Datos de Calibración).

2.Seleccionar la opción Informes >> Calibración de la Barra de Menús.

3.Sobre el diálogo de Opciones del Informe de Calibración mostrado:

·seleccionar la magnitud a calibrar

·seleccionar los puntos de medida a utilizar en el informe

4.Pulsar Aceptar para generar el informe.

 

Una vez creado el informe, se puede llamar de nuevo al diálogo de Opciones del Informe de Calibración, seleccionando Informes >> Opciones desde la Barra de Menús, o pulsando el botón de la Barra de Herramientas Estándar, mientras la ventana activa del entorno de trabajo de EPANET sea el citado informe.
