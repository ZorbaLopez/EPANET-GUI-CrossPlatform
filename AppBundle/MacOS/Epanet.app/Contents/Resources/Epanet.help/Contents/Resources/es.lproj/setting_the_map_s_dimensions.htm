Establecer las Dimensiones del Área de Dibujo

 Inicio  Anterior  Siguiente

Para fijar las dimensiones del área de dibujo:

1.Seleccionar Ver >> Dimensiones.
2.Introducir las nuevas dimensiones en el Diálogo de Dimensiones del Área de Dibujo mostrado o pulsar el botón Ajuste Automático para que EPANET calcule automáticamente las dimensiones a partir de las coordenadas de todos los objetos que incorpora la red.
3.Seleccionar las unidades de distancia a que están referidas las coordenadas introducidas.
4.Pulsar el botón Aceptar para redimensionar el esquema.

 

Nota: Si se va a utilizar un Mapa de Fondo con la opción de Cálculo Automático de la Longitud de las Tuberías, se recomienda fijar las coordenadas del área de dibujo inmediatamente después de crear un nuevo proyecto, ya que las unidades de medida de las distancias podrían diferir de las empleadas para medir la longitud de las tuberías. Estas últimas (metros o pies) dependen de que las unidades elegidas para medir el caudal sean métricas o US. EPANET convierte automáticamente las unidades si fuese necesario.
