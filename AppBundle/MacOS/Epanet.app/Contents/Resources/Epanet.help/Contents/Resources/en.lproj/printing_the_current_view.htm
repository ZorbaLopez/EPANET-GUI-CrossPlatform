Printing the Current View

 Top  Previous  Next

To print the contents of the current window being viewed:

Select File >> Print or click on the Standard Toolbar.

 

The following views can be printed:

} Data Browser (properties of the currently selected object)

}Network Map (at the current zoom level)

}Graphs (Time Series, Profile, Contour, and Frequency and System Flow plots)

}Tables (Network and Time Series tables)

}Reports (Status, Energy, Calibration, and Reaction)
