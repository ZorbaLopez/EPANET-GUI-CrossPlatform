Selección/Opciones de la Tabla

 Inicio  Anterior  Siguiente

El diálogo de Selección/Opciones de la Tabla se utiliza para personalizar el contenido de las tablas. La ventana contiene tres pestañas:

· Tipo

·Columnas

·Filtros

Cuando se crea una tabla, las páginas correspondientes a las tres pestañas están accesibles. Sin embargo, una vez la tabla creada, sólo aparecerán las pestañas correspondientes a Columnas y Filtros.
