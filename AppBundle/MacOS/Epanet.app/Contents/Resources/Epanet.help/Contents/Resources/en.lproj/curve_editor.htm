Curve Editor

 Top  Previous  Next

The Curve Editor edits the properties of a curve object. To use the Curve Editor:

1.Enter values for the following dialog items:

Item

Description

Curve ID

ID label of the curve (maximum of 31 characters

Description

Optional description of what the curve represents

Curve Type

Type of curve

X-Y Data

X-Y data points for the curve
2.As you move between cells in the X-Y data table (or press the Enter key) the curve is redrawn in the preview window.
3.Press OK to accept the curve.

 

You can also click the Load button to load in curve data that was previously saved to file or click the Save button to save the current curve's data to a file.
